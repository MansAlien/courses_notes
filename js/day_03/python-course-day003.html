Where to put the code

Let's explore where you can put your JavaScript code in an HTML document:

<head> Loads early, but may slow page rendering
→
<body> end Ensures HTML is loaded, but script loads last
→
External file Best practice: Use with defer attribute in <head>

Best Practice: External File with defer

<head>
    <script src="script.js" defer></script>
</head>

Loads early, runs after HTML is parsed

Improves page load performance

Keeps code organized and cacheable
