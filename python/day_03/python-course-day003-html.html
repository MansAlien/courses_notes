Python Course - Day 003

Dealing with data

Our apps contain code and data. Let's break it down:

Code
→
Manages Data
→
Stored in Memory

Data Categories:

Num

Age, Phone, Salary

String

Names, Event titles

Booleans

Yes/No conditions

Key Concepts:

  • Variables refer to data in memory
  • Code performs operations: Add, Edit, Delete

Data Types Overview

In Python, everything is an object. Let's explore some common data types:

int

Whole numbers

float

Decimal numbers

str

Text data

list

Ordered collection

tuple

Immutable collection

dict

Key-value pairs

bool

True or False

Example Code:

# Check types
print(type(10))        # int
print(type(10.9))      # float
print(type("hello"))   # str
print(type([1, 2, 3])) # list
print(type((1, 2, 3))) # tuple
print(type({"a": 1}))  # dict
print(type(True))      # bool
